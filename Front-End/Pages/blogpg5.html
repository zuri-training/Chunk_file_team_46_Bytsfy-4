Check undefined Javascript, two ways to do it.

Simple Answer would be to use If condition. there are two type of situation.

  1. In case of the variable is defined and not initialised or not assigned.
    Example 1:
    1.     var email;
    3.    if (email === undefined){
    4.     console.log("Email is undefined");
    5.     }

    Result:
    -------------------------
    Email is undefined

    Example 2:
    1.     var hadCoffee=false;
    3.    var isAwake;
    5.     if ( hadCoffee ){
    6.     isAwake=true;
    7.     }
    8.     //used ! to check the object is initilised or not
    9.    if ( !isAwake ) {
    10.     console.log("I hope not...");
    11.     }
    12.     if ( isAwake==undefined) {
    13.     }
    14.     console.log("I hope not...");
    15.     }

    Result:
    ---------------------------
    I hope not...
    I hope not...
    I hope not...

    In above case we have checked, if variable is defined but not initialised, there are three ways to verify that.

    using ” ! “
    using ” undefined “
    using ” null”

  2. In case of variable is not undefined.
    Example 1:
    1.     //email variable is not created at all
    3.    if(typeof email==='undefined' ){
    4.     console.log("Email is undefined");
    5.     }

    Result:
    -------------------------
    Email is undefined

    Example 2:
    1.     const actor = {
    2.       name: 'Tom Cruise',
    3.        photo:"Tom-Cruise.jpg"
    4.     }
    5.     if ( actor.name )
    6.     console.log("name:->"+actor.name);
    7.     }
    9.     if ( typeof actor.age === 'undefined' ){
    10.     console.log("Actor's age is undefined");
    11.     }

    Result:
    ---------------------------
    name:->Tom Cruise
    Actor's age is undefined

    “typeof” operator of JavaScript can be used to check the object is created or defined. It returns a string in case of the object is available.
