XML Pretty Print Using Python - With Examples

XML Pretty Print using Python is required frequently for testing, analyzing and debugging XML data. Python and XML both are treading in programming fields. Python generates dynamic XML string and received by the client. To save memory or bandwidth itâ€™s better to send the minified version of XML text.

Some time for debugging purposes, we need to see beautified XML data to understand and to Debug. This article will help to pretty-print XML data.

There are two examples in the post.

  1. Pretty Print XML String
  2. Pretty Print XML File
  1. XML Pretty Print using Python

    2. Here is an example:
    1. import xml.dom.minidom
    3. uglyxml = '<?xml version="1.0" encoding="UTF-8" ?><employees><employee><Name>Leonardo DiCaprio</Name></employee></employees>'
    5. xml = xml.dom.minidom.parseString(uglyxml)
    7. xml_pretty_str = xml.toprettyxml()
    9. print xml_pretty_str

    Result :
    1. < ?xml version="1.0" ? >
    2. < employees >
    3. < employees >
    4. < Name > Leonardo DiCaprio< /Name >
    5. < /employees >
    6. < /employees >


    Here is the explanation for the code.
    1. import xml.dom.minidom
      This is an XML library available in python to convert the DOM object from XML string or from the XML file.
    2. xml.dom.minidom.parseString
      parseString is a method which converts XML string to objectxml.dom.minidom.Document.
    3. xml.toprettyxml
      toprettyxml This method will indent the XML data and return as Pretty XML string.
